SetSelector Search Demo

This demonstrates the new search functionality for the CPU SetSelector component. The SetSelector allows searching through thousands of Pokemon trainer sets by either Pokemon species name or trainer name.

Features:

  • Search by Pokemon Species: Type "pikachu", "charizard", etc.
  • Search by Trainer Name: Type "champion", "youngster", "ace trainer", etc.
  • Case-insensitive: Search works regardless of capitalization
  • Real-time filtering: Results update as you type
  • Performance optimized: Only matching items are rendered
  • Persistent selection: Selected trainer ID is saved across sessions

• Type "youngster" to find all Youngster trainers
• Type "pikachu" to see all Pikachu sets
• Type "ace trainer" to find all Ace Trainer sets

Technical Implementation:

The search functionality filters the available sets in real-time as you type. The filtering logic checks both the Pokemon species name and the trainer name, making it easy to find exactly what you're looking for.
